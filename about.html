About This Blog

Welcome to jtgis! This is a minimal blog built with pure HTML, CSS, and JavaScript.

Features:

  • Courier New font for that retro terminal feel
  • Beige background for comfortable reading
  • Tag and date filtering
  • Pagination (10 posts per page)
  • GitHub-powered content management
  • Markdown support for post content

Built with love and minimal dependencies.
