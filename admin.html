## Video Embeds

For YouTube videos, use:
```
embed: "https://www.youtube.com/watch?v=VIDEO_ID"
embedType: "youtube"
```

For Vimeo videos, use:
```
embed: "https://vimeo.com/VIDEO_ID"
embedType: "vimeo"
```

For Spotify content, use:
```
embed: "https://open.spotify.com/episode/EPISODE_ID"
embedType: "spotify"
```
